Class NetworkPackageHandlerBase<T0, T1, T2>

implementation of INetworkPackageHandler. This is used to handle INetworkPackage instances received via a NetworkSubscriber. A single can not have multiple NetworkPackageHandlerBase<T>-Components attached that handle the same INetworkPackage type.
