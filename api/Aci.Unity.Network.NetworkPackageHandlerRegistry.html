Class NetworkPackageHandlerRegistry

A registry of INetworkPackageHandler instances. Used to assign received INetworkPackage instances to their correct handlers. Multiple Handlers can be registered for each INetworkPackage implementation. Handlers are identified via the packageType property which is compared to the type of incoming INetworkPackage instances.
