NetworkPackageHandlerRegistry

A registry of INetworkPackageHandler instances. Used to assign received INetworkPackage instances to their correct handlers. Multiple Handlers can be registered for each INetworkPackage implementation. Handlers are identified via the property which is compared to the type of incoming INetworkPackage instances.

NetworkPublisher

NetworkPublisher is a basic network communication client sending json messages via UDP. Messages will be serialized from class instances implementing INetworkPackage. Messages can be sent to multiple receivers by specifying one ip and one port per receiving client.

NetworkSubscriber

NetworkSubscriber is a basic network communication client receiving json messages via a UDP connection. Messages will be deserialized into class instances implementing INetworkPackage. To handle a new package type an instance of the type has to be bound via Zenject to INetworkPackage. Container.Bind{INetworkPackage}().To{ConcreteInterface}().FromNew(); Messages are handled via INetworkPackageHandler. Custom handlers can be derived from NetworkPackageHandlerBase<T>. They will then be automatically be added to a NetworkPackageHandlerRegistry via Zenject.
